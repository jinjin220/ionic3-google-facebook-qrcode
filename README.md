# ionic3-google-facebook-qrcode
This app was built with Ionic 3, Angular 4, Facebook and Google API and shows QRcode.

## Run
- `npm install`
- `npm install -g ionic`
- `npm install -g cordova`
- `ionic serve`

## Build iOS
- `ionic build`
- `ionic platform add iOS`

## Build Android
- `ionic build`
- `ionic platform add android`
